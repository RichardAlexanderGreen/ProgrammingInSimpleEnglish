Resource Manager

The run-time Resource Manager assigns tasks (actions) to Actors. An actor advertises the roles it can perform when it "checks-in" with the Resource Manager. When a running job invokes an action statement, the Resource Manager finds an available actor that can perform the role.

Single Threaded

Actors are assumed to be single-threaded and will only be assigned to one task at a time. The Resource Manager tracks which actors are busy versus which are available.
